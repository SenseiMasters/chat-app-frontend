## Initialization

Copy ".example.env" file to ".env.local" </br>
Then install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
